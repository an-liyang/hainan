## 项目包
* 项目地址：https://github.com/an-liyang/hainan.git
* 项目安装依赖：`npm install`
* 项目启动：`npm run dev`
* 构建：`npm run build`
* 补充说明：项目环境node，webpack等等请自行配置

---
## 项目主要目录介绍（参考vue-cli2）
* lang（多语言配置）
* router（路由配置）
* view(视图界面)
* util （常用方法）
 * data.js:接口输出文档
 * util.js:检验输出文档
